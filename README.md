# 🛒 Shopping Cart System

A full-stack e-commerce web application developed at CPUT (2025). The platform includes a persistent shopping cart, product management, and a structured database supporting multiple product categories.

## 🚀 Overview

This project focused on building a complete shopping experience, from browsing products to managing a cart and handling orders.

A key objective was implementing session-based state management to maintain cart data across multiple pages and user interactions.

## 🛠️ Technologies Used
**Frontend:** HTML5, CSS3  
**Backend:** PHP  
**Database:** MySQL / MariaDB  
**Concepts:** Session Management, Database Normalisation, Multi-page Navigation  

## ✨ Features
- 🛍️ Product listings across 3 categories  
- 🛒 Persistent shopping cart using PHP sessions  
- 📦 Order management functionality  
- 🗄️ Normalised database design (15+ products)  
- 🔗 Navigation across 8+ interconnected pages  

## 🧠 Key Contributions
- Designed and implemented a normalised MySQL database schema  
- Developed session-based cart functionality for persistent user experience  
- Built dynamic product and cart pages  
- Integrated backend logic with frontend interfaces
